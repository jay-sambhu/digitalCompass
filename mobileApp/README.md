# Digital Compass Mobile App

React Native + Expo mobile app for compass-based navigation and vastu-oriented compass modes.

## Features

- Live heading from device sensors
- Four compass modes:
  - Classic compass
  - 16-zone compass
  - 32-zone compass
  - Vastu chakra
- Camera tools route for compass capture workflows
- Expo Router-based navigation

## Project Structure

- `app/` route-based screens
- `components/` reusable UI and feature components
- `hooks/` sensor and app logic hooks
- `utils/` permissions, compass assets, and helper functions
- `styles/` shared style modules

## Run Locally

```bash
npm install
npm run start
```

Then open the app with Expo Go, Android emulator, or iOS simulator.

## Useful Scripts

```bash
npm run android
```
